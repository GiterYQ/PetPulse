2. 检查是否已有重复路线图：完成
3. 编写 MVP 技术实施路线图：完成
4. 更新文档索引和状态清单：完成
5. 复查文档质量：完成

## 约束

- 文档使用中文，技术名词、API 名、目录名保留英文。
- 不移动当前 React/Vite Demo 代码。
- 不决定最终一定使用 Flutter、React Native 或原生。
- 不引入新的依赖或后端代码。
- MVP 先验证互助闭环，不做支付、复杂认证、保险和 AI 医疗诊断。
