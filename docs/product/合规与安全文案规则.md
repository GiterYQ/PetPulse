# 合规与安全文案规则

## 1. 产品定位口径

PetPulse 是宠物急救互助协调工具，不是兽医诊断服务、救援担保服务或医疗机构。

推荐定位：

- 帮助用户更快发出求助。
- 帮助附近潜在帮助者看到请求。
- 帮助用户整理信息、发送提醒、跟踪响应。

## 2. 禁用文案

避免出现：

- `保证找到`
- `一定救活`
- `AI 诊断`
- `替代兽医`
- `官方救援保证`
- `精确定位公开给所有人`
- `认证后绝对安全`

## 3. 推荐替代表述

- `附近潜在帮助者`
- `建议尽快联系专业兽医`
- `AI 仅帮助整理信息`
- `结果取决于实际响应和线下情况`
- `默认仅显示大致区域`
- `你可以随时撤回或删除求助`

## 4. 场景规则

| 场景 | 文案要求 |
| --- | --- |
| 急救求助 | 强调紧急，但不制造恐慌 |
| 血液匹配 | 说明“潜在匹配”，不等于可直接输血 |
| 走失广播 | 保护地址，只展示必要范围 |
| AI 文案 | 用户确认后再发送 |
| 医疗内容 | 引导联系兽医，不给诊断 |
| 信任提示 | 写具体依据，不写绝对安全 |

## 5. 上线前检查

- [ ] 主流程无救援保证。
- [ ] AI 相关文案无医疗诊断暗示。
- [ ] 位置文案说明模糊范围和授权。
- [ ] 隐私政策覆盖宠物照片、位置、联系方式、通知 token。
- [ ] 删除求助或关闭账号的路径有产品设计。
